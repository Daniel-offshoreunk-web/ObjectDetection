# FTC Object Detection & Multi-Ball Tracking — 2026-27 Season

A lightweight, high-speed computer vision pipeline designed to run natively as a **Limelight 3A SnapScript** to track multiple game elements simultaneously. It utilizes an optimized color segmentation model combined with a geometric neural evaluation layer to filter background noise and provide smooth telemetry data over NetworkTables.

---

## Key Features
* **Natively Optimized for Limelight 3A:** Built around the required `runPipeline(image, ll_robot_inputs)` entry point for zero-lag coprocessor handling.
* **Persistent Multi-Ball Tracking:** Uses global scope variables to maintain a hysteresis tracking buffer, allowing the pipeline to remember ball positions across frame drops.
* **Centroid Velocity Vectors:** Matches balls between sequential frames using coordinate geometry distance optimization ($\sqrt{\Delta x^2 + \Delta y^2}$) to preserve target identities (`ID:0`, `ID:1`, etc.).
* **Target Locking Mechanics:** Features a required consecutive hit threshold before a target is sent to the robot drivetrain, completely filtering out brief background lighting glints.
